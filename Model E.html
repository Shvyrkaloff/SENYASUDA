Tesla Model 3

Модель 3 - это электрический четырехдверный седан среднего размера в стиле фастбэк, разработанный Tesla. Версия Model 3 Standard Range Plus обеспечивает полностью электрический запас хода по рейтингу EPA 262 мили (422 км), а версия Long Range - 353 мили ( 568 км). По словам Tesla, Model 3 имеет полностью автономное оборудование с периодическими обновлениями программного обеспечения, которые добавляют функциональность.

Ограниченное производство Model 3 началось в середине 2017 года, первый серийный автомобиль сошел с конвейера 7 июля 2017 года. Официальный запуск и поставка первых 30 автомобилей состоялись 28 июля. В октябре 2021 года Hertz car Rental заказала 100000 автомобилей Model 3 по полной цене для своего арендного парка.

Модель 3 продавалась как более доступная для большего числа людей, чем предыдущие модели Tesla. С начала 2020 года Model 3 является самым продаваемым электромобилем в мировой истории, а в июне 2021 года стал первым электромобилем, преодолевшим рубеж в 1 миллион мировых продаж. Model 3 стал самым продаваемым электромобилем в мире. автомобиль (PEV) в течение трех лет, с 2018 по 2020 год.
